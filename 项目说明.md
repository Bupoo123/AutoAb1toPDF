## 🚀 快速发布流程

### 方法一：使用 GitHub Actions 自动发布（推荐）

1. **推送 Git Tag**
   ```bash
   git tag v1.0.0
   git push origin v1.0.0
   ```

2. **自动完成**
   - GitHub Actions 会自动运行
   - 自动打包并创建 Release
   - 在 Actions 页面查看进度

### 方法二：手动创建 Release

1. **本地打包**
   ```bash
   ./package-release.sh 1.0.0
   ```

2. **在 GitHub 网页创建 Release**
   - 访问：`https://github.com/Bupoo123/AutoAb1toPDF/releases/new`
   - 上传 `AutoAb1toPDF-v1.0.0.zip` 文件
   - 填写发布说明

详细步骤请查看 [RELEASE_GUIDE.md](RELEASE_GUIDE.md)
